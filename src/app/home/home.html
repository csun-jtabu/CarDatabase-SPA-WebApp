Mom Loves You

This application is a Single Page Application (SPA) built with Angular on the Client Side.

The backend is powered by ASP.NET Core/C#.

Model-View-Controller (MVC) architecture is followed for both client and server sides.

The database used is Microsoft SQL Server.

The database contains a one-to-many relationship between two tables: Car Makes (1) and Car Models (*).

Authentication has been implemented to allow users to log in and access the tables.

Authorization has been implemented to restrict certain actions to Admin users only.

Regular Users can view car makes and their associated models.

Administrators can view, add, and delete entries as needed.

Login, Add, and Delete functionalities utilized Reactive Forms for complex data entry.

Navigation Overview:

Make List: View all car makes currently in the database. You can see the number of models associated with each make. Also, click on a make to view the number of models it has.

Model List: Check the models associated with each car make. This helps track which models belong to which make.

Add New Make: Add a new car make to the database. Admin login is required.

Delete a Make: Remove a car make from the database. Admin login is required.

Tips / Requirements:

  • Admin login is required for adding or deleting car makes.
  • All form fields have validation rules to prevent empty or invalid entries.
  • The system ensures that models are associated correctly with their makes.
